Project Overview

This lab demonstrates the use of SQL views, stored procedures, and functions to manage and manipulate data within an inventory database.

Download Document

Step by Step Process

  • Setup MySQL Workbench: Start MySQL server, open Workbench, and use democodes.sql file for queries.
  • Use Inventory Database: Load and use inventory.sql.
  • Create Views, Stored Procedure, and Function:
    • Views for product filtering and computation.
    • Stored procedure to update vendor name.
    • Function to display product info based on vendor code and state.
  • Execute and document the results with screenshots.
